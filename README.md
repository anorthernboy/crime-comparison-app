# crime comparison app

A fully functional, responsive crime comparison app.

Enter two postcodes to compare crime levels overall and delve into crimes by category

## Getting Started

### Prerequisites

- Node.js

To get the project up and running locally:

- Clone this repo and then run the command `npm install` from the root directory. This will install all the necessary dependencies.
- Run the command `npm start` to start a server. This should automatically open the crime comparison app in your browser, but you can also navigate to the URL given in the console.
